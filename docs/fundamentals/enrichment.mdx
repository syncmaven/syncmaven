<Warning>
  Enrichment is a feature that is in active development and is not yet available for
  production use.
</Warning>

Enrichment is a service that allows you to enrich your data with additional information from external sources.
It's an optional part of the [sync](/fundamentals/sync) process.

Enrichments doesn't have its own place to define it. It's defined in [sync](/fundamentals/sync)
file as a [connection](/fundamentals/connection) to a service and optional parameters.

Please refer to the [sync](/fundamentals/sync) documentation for more information.
